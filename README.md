JatreNammaPride 🌟
---
JatreNammaPride is an Android-based cultural assistance application developed to digitally showcase Karnataka’s traditional fairs and local heritage. The application combines modern mobile technology with regional culture to provide users with an engaging and informative experience.
The app includes smart AI-powered guidance, multilingual interaction, festival information, and interactive user features designed to improve accessibility and user engagement.

✨ Key Features
---
Smart AI-based cultural assistant
Festival and heritage information access
Kannada and English language support
Modern and responsive mobile interface
Interactive user navigation experience
Real-time Firebase integration
Location-based support features
Simple and clean user experience


🛠 Technologies Used
---
1.Kotlin
2.Android Studio
3.Jetpack Compose
4.Firebase Realtime Database
5.Google Gemini API
6.MVVM Architecture
7.JSON Parsing

📱 Application Modules
---
Splash Screen
Home Dashboard
AI Guide Assistant
Cultural Information Section
Location and Navigation Features
Recommendation System


🚀 Installation Steps
---
1. Clone Repository
git clone https://github.com/your-username/JatreNammaPride.git

2. Open in Android Studio
Open the extracted project folder using Android Studio.

3. Firebase Setup
Download the Firebase configuration file:
google-services.json
Place it inside:
app/

4. Sync Gradle
Allow Android Studio to complete Gradle synchronization.

5. Run Application
Connect an Android device or emulator and run the project.
